Nuestro proyecto se trata de un sistema de gestion de proyectos para la empresa Konstructiva


Se trata de 5 modulos los cuales son:

Modulo Proyecto - presupuesto : crear y modificar los proyectos y presupuestos de cada uno de estos. 

Modulo Cliente : crear, modificar y eliminar los clientes (un cliente se define como la persona que encarga un proyecto) 

Modulo Ejecución del proyecto : crear y modificar ítems de cada proyecto 

Modulo Personal : aparte del personal temporal, existen personal de planta, en este modulo se puede crear y modificar informacion de estos, como el contrato, la afiliacion a la eps y los datos basicos. 

Modulo Contable: recibe los datos del proyecto y la demas información relevante para la parte contable de la empresa, como gastos en materiales y mano de obra, ganancias y perdidas.


Ademas de esto, en nuestro sistema habran 3 perfiles diferentes para el manejo de la aplicacion. 

Perfil Administrativo : es el administrador de la empresa, este verá todos los modulos de la aplicación.

 Perfil Compras: es la persona encargada de realizar las compras de materiales, este vera el modulo de ejecucion del proyecto, y podra subir informacion de las compras, como, valor de la factura, de donde la compro, fecha y a qué proyecto pertenece esta compra. 

 Perfil Contable: es la persona que lleva la contabilidad de la empresa, esta vera solo el modulo contable, donde se encuentra toda la información relacionada al dinero de la empresa.
